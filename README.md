ANEMIA DETECTION


We developed three parallel deep learning pipelines for non-invasive anemia detection across fingernail, palpebral conjunctiva, and retinal image modalities, enabling a comprehensive multi-modal comparison study not commonly found in existing literature. Our fingernail model (EfficientNetB0, ~4,260 images) achieved a strong 93% baseline accuracy, while the conjunctiva pipeline (CP-AnemiC dataset, ~710–730 images) used two-stage transfer learning with StratifiedGroupKFold cross-validation for robust generalization, and the retinal pipeline (MobileNetV3-Large, ~800 images) employed 5-fold CV with OOF threshold tuning for optimized classification boundaries. Our key novelty lies in integrating Grad-CAM explainability across all three modalities, making the models clinically interpretable rather than black-box, and in designing a CNN + color-feature fusion approach for the conjunctiva branch to combine learned and handcrafted hemoglobin-relevant features. We resolved several real-world data challenges (e.g., locale-based decimal formatting in labels, dataset filtering) to ensure clean, reliable training pipelines across all three datasets. This positions the project well against the benchmark reference accuracy of 99.12% (Asare et al., 2023), with our fingernail modality already showing competitive performance and the conjunctiva/retinal modalities benefiting from ongoing architecture refinement.
